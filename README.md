# trashify
Linux utility for trashing files

## Installation
1. Clone the repository
   ```
   git clone https://github.com/Sky1sBloo/trashify.git
   ```
2. Generate CMake files
   ```
   cmake ./
   ```
3. Build
   ```
   cmake --build
   ```
## Usage
```
--help  |  -h
```
Lists of commands
```
--trash [file]  |  -t
```
Trash a specific file or directory
