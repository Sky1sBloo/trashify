# trashify
Linux utility for trashing files

## Installation
1. Clone the repository
   ```
   git clone https://github.com/Sky1sBloo/trashify.git
   ```
2. Go to Debug directory
   ```
   cd Debug
   ```
3. Generate CMake files
   ```
   cmake ../
   ```
4. Build
   ```
   cmake --build
   ```
## Usage
```
--help  |  -h
```
Lists of commands
```
--trash [file]  |  -t
```
Trash a specific file or directory
